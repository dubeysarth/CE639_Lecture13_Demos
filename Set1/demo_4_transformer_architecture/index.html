Parameters & Architecture

Adjust global parameters — all layer shapes update instantly.

Try 1: Cross-Attention

Switch to Enc-Decoder. Click the "Cross-attention" layer — see how decoder queries (T×d) attend to encoder keys (L×d). This is where the model reads historical streamflow memory.

Try 2: Causal Mask

In Decoder-Only, click "Causal self-attention". The attention matrix is lower-triangular — position i cannot see j > i. This enforces physical causality in the forecast.

Try 3: Scale L

Increase L from 24 to 192. Watch the attention complexity grow as O(L²). In the Informer tab, ProbSparse reduces this to O(L log L) — critical for long streamflow records.
