Multi-Head Attention & Head Specialization

Multiple attention heads process the same 28-day streamflow sequence simultaneously, each learning to focus on a different hydrological pattern — flood onset, baseflow recession, seasonal trend, and rainfall lag. Click any head panel to expand it.

Why Multiple Heads?
A single head must trade off between capturing the flash-flood spike (short-range, high-magnitude) and the slow soil-moisture memory (long-range, diffuse). With 4 heads, each head specializes on a different time-scale simultaneously — they're concatenated to form the full representation.

Try 1: Single vs. Multiple Heads

Set h=1. The single head must average over all patterns. Switch to h=4 — notice each head now isolates a distinct temporal pattern (spike, recession, seasonal, lag).

Try 2: Increase d_model

Increase d_model from 16 to 128. Each head now has more capacity (d_k = d_model/h) — the attention maps sharpen and show finer structure in the flood event.

Try 3: Click a Head

Click any head panel to expand it to full size. Read the specialization label and note which days receive the highest attention weight (darkest cells).
